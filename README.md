#### 6. 技术风险
> Python 功能强大，但相关的技术也有风险。  
> **忠告：不要用爬虫技术**  
>>1.爬虫容易搞坏网站。（刑法二百八十六条，破坏计算机信息系统罪）  
>>2.爬虫很容易采集个人信息。（网络信息安全法，GDPR等法律相关条款）  
>>3.爬虫很容易采集商业机密和国家机密等。
